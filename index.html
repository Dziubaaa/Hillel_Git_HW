Basic Git Commands

  • git clone <repository URL>: Create a copy of a remote repository on your local machine.
  • git add <file>: Stage changes to a file for the next commit.
  • git commit -m "Commit message": Commit staged changes with a descriptive message.
  • git status: Check the status of your working directory and see which files are staged, modified, or untracked.
  • git log: View a log of all previous commits in the repository.
  • git branch: List all branches in the repository and highlight the current branch.
  • git merge <branch name>: Merge changes from another branch into the current branch.
  • git pull: Fetch and merge changes from a remote repository into the current branch.
  • git push: Push local commits to a remote repository.
  • git init: Initialize a new Git repository in the current directory.
